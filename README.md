# Task 23

A responsive landing page prototype built using Tailwind CSS in the browser.

## Overview

This task demonstrates a simple responsive hero section with navigation, call-to-action buttons, and feature cards.

## Features

- Responsive layout using Tailwind utility classes
- Sticky header with navigation and CTA button
- Hero section with a headline, description, and image
- Feature cards section with icon highlights
- Footer with branding and copyright notice
- Mobile-friendly design using responsive classes

## How to view

1. Open `index.html` in your browser.
2. Alternatively, use a local web server for a better development experience.

## Notes

- The page uses the Tailwind CDN via `@tailwindcss/browser`.
- Icon support is provided by Ionicons.
- The design is intended as a landing page hero section, not a full website.
